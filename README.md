# LCS Warrior AI

Liberty Christian School's AI platform — skills, bootstrap scripts, and auto-discovery hooks for the [Warriors Wiki](https://lcs-wiki-bpd1iwhcgswk.adom.cloud/).

## Structure

```
skills/           # Claude Code skills (installed to ~/.claude/skills/)
bootstrap/        # Container bootstrap/unbootstrap scripts + welcome page
hooks/            # Auto-discovery hooks (installed to ~/.lcs/hooks/)
```

## Bootstrap

Set up a fresh Adom container for LCS:

```bash
curl -fsSL https://lcs-wiki-bpd1iwhcgswk.adom.cloud/static/apps/lcs-bootstrap/bootstrap.sh | bash
```
