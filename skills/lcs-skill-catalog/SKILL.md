# LCS Skill Catalog

Skills available for Liberty Christian School. Each skill teaches
Claude how to help with a specific school task.

## School Identity and Design

| Skill | Description |
|---|---|
| `lcs-brand` | Liberty Christian School brand identity -- colors (navy #002855, gold #C5A44E), typography, logo usage, and design system tokens |
| `lcs-ui-patterns` | Non-negotiable UI rules for every LCS app. Tooltips, hover states, button feedback, accessibility, mobile responsiveness |
| `lcs-app-header` | The canonical Warriors-branded header bar for all LCS webview apps |

## Communication

| Skill | Description |
|---|---|
| `lcs-gchat` | Post to Google Chat spaces (#teachers, #students, #parents, #strive-center) from the LCS container |

## Wiki and Publishing

| Skill | Description |
|---|---|

## AI and Programming
